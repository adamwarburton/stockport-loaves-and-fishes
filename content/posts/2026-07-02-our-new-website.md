Welcome to the new home of Stockport Loaves and Fishes on the internet. Same charity, same Sunday dinners, considerably fewer cobwebs.

Everything you need is a tap away: [what to expect if you need help](/help), [what we need this week](/how-to-help), and [our story](/our-story) — which starts, as all good stories do, with a car boot full of hot food.

The site is run by the same volunteers who run the kitchen, so if you spot something wrong, [tell us nicely](/contact) and we'll fix it between shifts.
